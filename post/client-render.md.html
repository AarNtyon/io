client-render 客户端渲染

Author: Rain 写于 2023-06-10 11:31:25

遇到客户端在组件渲染正常，控制台却报错的问题

window is not defined

const ClientComponent = dynamic(() => import('@/components/ClientComponent'), {
    ssr: false,
})

next/dynamic 是 Next.js 的一个高级组件，可以使得组件能够在客户端进行动态加载，而不是在服务端渲染。
